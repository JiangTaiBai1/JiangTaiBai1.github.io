## 项目概述

这是一个面向中小型团队的任务管理 SaaS 平台，解决传统项目管理工具配置复杂、成本高昂的问题。

## 核心功能

- **看板视图**：拖拽式任务管理
- **甘特图**：项目时间线可视化
- **实时协作**：基于 WebSocket 的多人同步
- **多租户**：完整的团队隔离和权限系统

## 技术选型

| 层级 | 技术 |
|------|------|
| 前端 | React 18 + TypeScript |
| 后端 | Node.js + Express |
| 数据库 | PostgreSQL + Redis |
| 实时通信 | Socket.io |
| 部署 | Docker + Railway |

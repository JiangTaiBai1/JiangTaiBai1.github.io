汽车交易展示站

课程综合项目，使用原生 HTML + CSS 搭建的汽车交易展示页面，包含大图 Banner、库存筛选、车辆卡片与营销落地页结构。

2026/4/20
HTML · CSS · 静态页面 · 课程项目
在线演示

项目简介

这是一个偏视觉呈现型的静态展示站，主题围绕汽车交易与车型推荐展开，重点训练页面拆分、图文排版、横幅区设计与复杂素材的组合布局。

我做了什么

  • 完成首页大图展示区与品牌视觉包装
  • 搭建车辆推荐列表与价格信息卡片
  • 实现多段式营销展示结构与页脚信息区
  • 使用大量切图素材还原课程设计稿

技术点

  • 原生 HTML 页面结构组织
  • CSS 定位、背景图、分区布局
  • 多素材页面整合与视觉还原
